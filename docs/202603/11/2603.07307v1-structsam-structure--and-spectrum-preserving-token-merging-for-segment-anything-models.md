## 摘要
近期针对视觉 Transformer (ViT) 的 Token 合并技术通过减少自注意力机制处理的 Token 数量，在通常无需重新训练的情况下实现了显著的加速。然而，将这些技术直接应用于 Segment Anything Model (SAM) 系列并非易事：SAM 的图像编码器混合了窗口注意力和全局注意力，且其掩码解码器依赖于密集的、受提示词（prompt）调节的特征来进行精确的边界预测。我们在严格的开箱即用（off-the-shelf）设置下，系统地评估了 SAM 和 Medical SAM 上的代表性 Token 合并方法，发现现有的目标选择启发式算法在合并率增加时会侵蚀边界并泄露提示信息。我们提出了 StructSAM，这是一个专为 SAM 定制的保持分辨率的合并-拆解（merge-unmerge）框架。StructSAM 从一阶特征梯度中计算轻量级的 Token 能量评分，利用基于网格的平坦度筛选来保护边界和提示区域，并将平坦区域内的 Token 合并至低能量目标，同时进行显式的 Token 恢复。我们进一步提供了一个谱图粗糙化（spectral graph coarsening）视角，证明与随机或窗口限制的基准方法相比，基于评分引导的合并能产生有界的拉普拉斯谱失真。在八个自然和医学基准测试中，StructSAM 在 mIoU/Dice 仅轻微下降的情况下，将编码器 FLOPs 降低了 25-30%（结合提示感知合并可达 40% 以上），在相同计算量下持续优于 ToMe、PiToMe、ToMeSD、VidToMe 和 ALGM。

## Abstract
Recent token merging techniques for Vision Transformers (ViTs) provide substantial speedups by reducing the number of tokens processed by self-attention, often without retraining. However, their direct application to the Segment Anything Model (SAM) family is nontrivial: SAM's image encoder mixes windowed and global attention, and its mask decoder relies on dense, prompt-conditioned features for precise boundary prediction. We systematically evaluate representative token-merging methods on SAM and Medical SAM in a strict off-the-shelf setting, and find that existing destination-selection heuristics can erode boundaries and leak prompt information as merge rates increase. We propose \textbf{StructSAM}, a resolution-preserving merge-unmerge framework tailored to SAM. StructSAM computes a lightweight token-energy score from first-order feature gradients, uses grid-based flatness screening to protect boundary and prompt regions, and merges tokens within flat areas toward low-energy destinations with explicit token recovery. We further provide a spectral graph coarsening view showing that score-guided merging yields bounded Laplacian spectral distortion compared to random or window-restricted baselines. Across eight natural and medical benchmarks, StructSAM reduces encoder FLOPs by 25-30\% (up to 40\%+ with prompt-aware merging) with minor drops in mIoU/Dice, consistently outperforming ToMe, PiToMe, ToMeSD, VidToMe, and ALGM at the same compute.

#### 1. 核心问题与整体含义
Segment Anything Model (SAM) 及其变体（如 MedSAM）在视觉分割领域表现卓越，但其计算开销巨大，尤其是图像编码器（Encoder）占据了超过 98% 的参数和计算量。现有的 Token 合并技术（如 ToMe）虽能加速 Vision Transformer，但直接用于 SAM 时存在两个核心矛盾：
*   **结构破坏**：SAM 的解码器需要高分辨率的密集特征来预测精确边界，而传统合并方法会导致边界模糊。
*   **提示词失效**：现有的合并策略可能导致提示词（Prompt）覆盖区域的信息丢失。
本文旨在提出一种**无需重新训练、开箱即用**的 Token 合并框架，在显著降低计算成本的同时，保持 SAM 的分割精度和边界细节。

#### 2. 方法论
StructSAM 提出了一个“合并-计算-还原”（Merge-Compute-Unmerge）的流程，核心思想是利用图像特征的结构化信息指导 Token 的精简：
*   **特征梯度能量估计**：利用一阶有限差分（如 Sobel 算子）计算特征图的梯度幅值。高梯度区域被视为“能量高”的边界区域，低梯度区域则视为平坦背景。
*   **网格平坦度筛选**：将 Token 划分为 $s \times s$ 的网格单元（Cell）。计算每个单元的平坦度得分（基于单元内最大梯度值）。
*   **结构感知合并**：
    *   **保护区**：高梯度单元（包含边界）被完全保留。
    *   **合并区**：平坦单元内的 Token 被合并至该单元内梯度最低的“目标 Token”。
*   **显式 Token 恢复（Unmerge）**：在自注意力计算完成后，将合并后的特征复制回原始分辨率的网格中，以满足 SAM 解码器对密集特征的需求。
*   **提示感知变体**：若存在框提示（Box Prompt），则降低框内区域的合并率，提高框外背景的合并率。
*   **理论支撑**：从谱图论角度证明，这种基于评分引导的合并策略相比随机合并，能显著降低拉普拉斯矩阵的谱失真（Spectral Distortion）。
